# 🎬 Movie Analytics Platform

A modern movie analytics web application built using a clean, scalable frontend architecture. This project focuses on delivering real-time movie insights such as trending, popular, and top-rated films while maintaining a modular and maintainable codebase.

---

## 🚀 Features (Currently Implemented)

### 🎥 Movie Discovery

* 🔥 Trending movies
* ⭐ Top-rated movies
* 📈 Popular movies

### 🔎 Search Functionality

* Search movies dynamically using user queries
* Real-time data fetching and rendering

### 🎭 Genre Filtering

* Filter movies based on genres
* Dynamic genre list handling

### 📄 Movie Details Page

* Detailed information for each movie
* Structured UI for readability

### 🌐 Language Support

* Multi-language handling (if applicable)

### ⚛️ Scalable Architecture

* Custom hooks for API abstraction
* Context API for global state management
* Modular folder structure

---

## 🧠 Tech Stack

* **Frontend:** React (with Hooks)
* **State Management:** Context API
* **API:** TMDB (The Movie Database)

---

## 🏗️ Project Structure

```
src/
```

## ⚠️ Limitations (Current State)

* ❌ No backend (direct API consumption)
* ❌ No data persistence
* ❌ No authentication system
* ❌ No user personalization

```

```
